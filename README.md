# opsadmin
web shell terminal 资产收集 录像审计

本项目是本人兴趣想要做的东西,可能之前代码经验不丰富,代码文件和变量命令有点问题,之后会改回来的.还有本平台只是初成.有一些功能还在完善中,但是.
基本的资产生成,资产收集,还有基于web上shell terminal,还有在web shell terminal上录像审计功能是可以的.本项目也是借鉴几个开源项目的代码来完成，
 前端也是做得不太好.。日后有时间进行改进.
